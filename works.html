UnunChat - リアルタイムチャットアプリ

Socket.IOを使った双方向通信型のチャットアプリ。Node.jsとExpressで構築し、Herokuへデプロイ。 チャット履歴はSQLite3で保存（本番環境への移行も想定）。

使用技術
  • フロントエンド:
    • HTML/CSS：チャットUIの設計
    • JavaScript：動的なメッセージ表示、タイムスタンプの表示
    • Socket.IO（クライアント）：リアルタイム通信処理
  • バックエンド:
    • Node.js + Express：サーバー構築とAPIルーティング
    • Socket.IO（サーバー）：リアルタイムイベント処理
    • SQLite3：チャット履歴の保存と取得
  • インフラ・その他:
    • Heroku：アプリのデプロイと本番環境運用
    • Git / GitHub：ソース管理とデプロイ連携
学んだこと
  • 双方向通信（Socket.IO）と非同期通信（HTTP通信）の違いの理解
  • Node.jsとExpressを用いたサーバー開発の基本
  • Herokuへのデプロイ方法と環境変数の設定
  • フロントとバックをつなげたシンプルなフルスタック開発体験
  • トラブル対応力（404エラー、ポート競合、CORS、DB接続など）
